**Блоки кода**
Встроенный (inline) код:
Команда `git status` показывает состояние репозитория.
Многострочный код:

```python
print("Hello, Markdown!")
```
С указанием языка программирования:
**Python**: python
```py
def hello():
    print("Hello from Python!")
    return True
```
**JavaScript**: javascript
```js
function hello() {
    console.log("Hello from JavaScript!");
    return true;
}
```
**Bash (терминальные команды)**: bash
```bash
git add .
git commit -m "Update"
git push origin main
```
**C#**: csharp
```
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Hello from C#!");
    }
}
```
**HTML**: html
```html
<!DOCTYPE html>
<html>
<head>
    <title>Пример HTML</title>
</head>
<body>
    <h1>Hello, World!</h1>
</body>
</html>
```
**Цитаты**
Простая цитата:

>Это цитата в Markdown.

Многострочная цитата:

>    Это первая строка цитаты.
>    Это вторая строка той же цитаты.
>    Это продолжение цитаты.

Вложенные цитаты:

>Первый уровень цитирования
>>Второй уровень (вложенная цитата)
>>>Третий уровень вложенности

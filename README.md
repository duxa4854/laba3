# Лабораторная работа по системам контроля версий и Markdown

## Краткое описание
Данная лабораторная работа направлена на освоение современных инструментов разработки. В рамках работы изучаются системы контроля версий Git, язык разметки Markdown для оформления документации, а также основы математических формул с использованием LaTeX. Проект демонстрирует навыки работы с распределёнными системами разработки и оформления технической документации.
